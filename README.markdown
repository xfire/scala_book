scala experiments
=================

here are some tests and experiments while digging deeper into the bowels of scala.


building and running
--------------------

### requisites

    - installed [jdk](http://java.sun.com/)
    - [sbt](http://code.google.com/p/simple-build-tool/) >= 0.6.12

### build

    > sbt build

### run

    > sbt run
